注意

本文档只介绍Yearning相关安装配置，对于Inceotion相关配置需自行添加。如只配置了Yearning却没有配置Inception则仍然无法正常工作。所以请确保自己已熟读inception相关文档https://github.com/cookieY/inception-document。

如有任何问题 请查看 相关问题页面 获取Q&A 或在群共享内下载Q&A.pdf

特别提醒：

测试及使用时请确保使用chrome最新版本浏览器，请勿多开网页窗口测试yearning，以免导致未知问题。

v1.3.0及以上版本安装

从v1.3.0版本开始将采取docker运行的方式，不再提供普通安装教程

安装：

注意:inception安装不在本安装教程之内，请自行安装inceotion，可使用install目录下inception.tar解压后获得的inception二进制直接运行

1.下载Yearning,进入install/yearning-docker-compose目录

docker-compose up -d

请注意本地不要占用8080 如需要更改端口可再docker-compose.yml文件中更改，3306端口不可更改！

之后访问8080端口。

默认用户admin 密码: Yearning_admin

登陆后请通过设置页面设置inception及其他配置信息

使用外部mysql

docker-compose中已经包含了mysql且挂载本地目录，不存在容器销毁数据消失的问题，但如果你还是想使用外部mysql,可使用以下步骤：

1. 新建Yearning库，设置字符集为UTF-8，然后在Yearning库中导入sql文件(Yearning/install/yearning-docker-compose/init-sql)

2. docker run -d -e HOST=主机地址 -e MYSQL_ADDR=mysql地址 -e MYSQL_USER=mysql用户名 -e MYSQL_PASSWORD=mysql密码 -p8080:8000 registry.cn-hangzhou.aliyuncs.com/cookie/yearning:latest

注意事项

默认超级管理用户只拥有管理页面的访问权限,其他的权限需自行增加！详情请查看使用说明用户管理
